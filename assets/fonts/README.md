# Fonts folder

Drop these two files here from Google Fonts before pushing:

1. DancingScript-VariableFont_wght.ttf
   - Get from: https://fonts.google.com/specimen/Dancing+Script
   - Click "Get font" → "Download all" → unzip → look for DancingScript-VariableFont_wght.ttf

2. Montserrat-VariableFont_wght.ttf
   - Get from: https://fonts.google.com/specimen/Montserrat
   - Click "Get font" → "Download all" → unzip → look for Montserrat-VariableFont_wght.ttf

Once both files are in this folder, you can delete this README.
